## ✨ Overview

A clean, modern UI design for an **appointment booking application**, crafted for beauty and wellness businesses. It lets customers browse services, pick a convenient date and time, and confirm their bookings in just a few taps — no friction, no confusion.

## 🖥️ Key Screens & Features

| Screen | What it does |
|---|---|
| 💅 **Service Selection** | Browse available services with clear pricing & duration |
| 📅 **Date & Time Scheduling** | Interactive calendar + time-slot picker |
| 🧾 **Booking Summary** | Review service, date, time & price before confirming |
| ✅ **Booking Confirmation** | Clean success screen once the appointment is booked |
| 📱 **Responsive Layout** | Adapts smoothly across screen sizes |

## 🎨 Design Highlights

- 🧘 Minimal, whitespace-driven layout that keeps focus on the booking flow
- 🧭 Clear visual hierarchy guiding users step-by-step
- 🎯 Consistent, friendly components suited to beauty & wellness branding
- 🌊 Smooth, guided journey from browsing → confirmation
